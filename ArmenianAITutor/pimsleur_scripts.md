# Pimsleur-Style Audio Lessons: Scripts for Lessons 1-20
# Western Armenian — 10-Minute Format
# Status: DRAFT — Requires native speaker validation

## FORMAT GUIDE

Each lesson follows this structure:
- **[INSTRUCTOR]** — English narration (male voice, calm and clear)
- **[MALE]** — Armenian phrase spoken by male voice (Hayk)
- **[FEMALE]** — Armenian phrase spoken by female voice (Anahit)
- **[PAUSE X]** — Silence for X seconds (learner responds)
- **[SFX]** — Sound effect (chime for new section, etc.)

Phonetic pronunciation is in parentheses after Armenian text.
Items marked with ⚠️ need native speaker validation.
